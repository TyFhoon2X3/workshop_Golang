# Workshop Golang

โปรเจกต์สำหรับเรียนรู้และฝึกฝนภาษา Go (Golang)

## ไฟล์ในโปรเจกต์

| ไฟล์/โฟลเดอร์ | รายละเอียด |
|--------------|------------|
| `go.mod` | ไฟล์กำหนดโมดูลและเวอร์ชันของ Go |
| `grade.go` | โปรแกรมคำนวณเกรดและหน่วยกิต จากคะแนนที่กรอก |
| `test.go` | โปรแกรมรับชื่อวิชาและคะแนน แล้วแสดงผลเกรด |
| `README.md` | ไฟล์เอกสารอธิบายโปรเจกต์ |
| `member/main.go` | ไฟล์ทดสอบในโฟลเดอร์ member |
| `struct/product.go` | ไฟล์เก็บโครงสร้างข้อมูล product |

## วิธีรันโปรแกรม

### รัน grade.go
```bash
go run grade.go
```

### รัน test.go
```bash
go run test.go
```

## สิ่งที่ได้เรียนรู้

- การใช้งาน `fmt.Scan` สำหรับรับข้อมูลจากผู้ใช้
- การใช้งาน `if-else` สำหรับตรวจสอบเงื่อนไข
- การใช้งาน `for` loop
- การสร้าง slice และการเพิ่มข้อมูล
- การคืนค่าจากฟังก์ชัน

## ข้อกำหนด
