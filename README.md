# Workshop Golang

โปรเจกต์สำหรับเรียนรู้และฝึกฝนภาษา Go (Golang)

## ไฟล์ในโปรเจกต์

| ไฟล์ | รายละเอียด |
|------|------------|
| `grade.go` | โปรแกรมคำนวณเกรดและหน่วยกิต จากคะแนนที่กรอก |
| `test.go` | โปรแกรมทดสอบตรวจสอบเลขคู่และเลขคี่ (1-10) |
| `go.mod` | ไฟล์กำหนดโมดูลและเวอร์ชันของ Go |

## วิธีรันโปรแกรม

### รัน grade.go
```bash
go run grade.go
```

### รัน test.go
```bash
go run test.go
```

## สิ่งที่ได้เรียนรู้

- การใช้งาน `fmt.Scan` สำหรับรับข้อมูลจากผู้ใช้
- การใช้งาน `if-else` สำหรับตรวจสอบเงื่อนไข
- การใช้งาน `for` loop
- การสร้าง slice และการเพิ่มข้อมูล
- การคืนค่าจากฟังก์ชัน

## ข้อกำหนด

- Go 1.18 หรือเวอร์ชันที่ใหม่กว่า
